TOTONOI LP 初期コード

【入っているファイル】
- index.html
- style.css
- script.js

【使い方】
1. GitHubの totonoi-lp フォルダ直下に、この3ファイルを入れてください。
2. 画像23枚は images フォルダに入れてください。
3. 予約URLを変更する場合は script.js の RESERVATION_URL を書き換えてください。
4. GitHub Desktopで Commit → Push してください。
